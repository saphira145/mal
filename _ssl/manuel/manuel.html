ログインする

管理画面の利用にはログインが必要です。
一般権限のユーザーはログインできません。

  1. 管理画面を開く
    ブラウザで以下の URL にアクセスします。
    https://www.mal-marche.com/mald/sagacite/admin/
    ログイン画面が開きます。
  2. ID / パスワード入力
    ID と パスワードを入力し、「ログイン」ボタンをクリックします。
  3. ログイン完了
    ログインに成功すると、管理画面トップページが開きます。

整合性チェック

本番リリースの前に、諸元情報と各種設定した内容の整合性を確認します。
緊急度の高い問題は、赤文字で表示されます。

  1. 上部メニューの「トップ」をクリックします。
    「トップ」画面が開きます。
  2. 整合性チェックを行う
    「チェック」ボタンをクリックします。
    チェック結果が表示されます。
  3. データを修正する
    チェック結果に従い、該当データを修正します。

車種を設定する

  • 作業者 三菱オートリース様 ホーユー
  • 設定対象 一般 営業
  • 反映タイミング 要リリース

設定する項目は以下の通りです。

  • 代表車種グレード
  • 手動バリュー
  • お値打ちスコア (キャンペーンスコア)
  • Webカタログ
  • 写真ギャラリー

他の項目は、ホーユーが登録・変更します。
また車種を改廃する場合は、諸元情報と連動させる必要があるため、ホーユーにお問い合わせください。

代表車種グレードについて

  • 代表車種グレードは以下の用途で使用します。
    • 比較画面の初期表示
    • 自動バリュー (参照)の諸元判定
  • 代表車種グレードは以下の3グレードを設定できます。
    • 標準2WD
    • 標準4WD
    • 最廉価
  • 代表車種グレードが1つも設定されていない場合、その車種はサイトに掲載されません。
    • まだ代表車種グレードを設定していない車種 (新発売)
    • 車種のグレード構成が変更され、代表車種グレードに設定していたグレードが無くなった車種 (モデルチェンジ)
  • 車種諸元データが更新された後、本番リリース前には必ず「整合性チェック」を行ない、設定漏れが無いかを確認してください。

代表車種グレードを設定する

代表車種グレードは、「車種管理」画面より設定します。

  1. 上部メニューの「車種管理」をクリックします。
    「車種管理」画面が開きます。
  2. 車種詳細画面を開く
    設定したい車種を検索し、検索結果一覧の「詳細」ボタンをクリックします。
    「車種詳細」画面が開きます。
  3. 車種編集画面を開く
    車種詳細画面の「編集」ボタンをクリックします。
    編集画面が開きます。
  4. 代表車種グレードを設定する
    「代表車種グレード」を以下のように設定します。
    • 「最廉価」は、必ず設定します。
    • 「標準2WD」「標準4WD」は、いずれかを必ず設定します。
    • グレード数が少ない場合、「標準2WD」と「最廉価」、または「標準4WD」と「最廉価」は同一グレードを設定しても構いません。
  5. 確認する
    内容を変更した後、「更新する」ボタンをクリックします。
    確認画面が表示されますので、内容に問題がなければ、「更新する」ボタンをクリックします。
  6. 更新完了
    更新完了画面が表示されます。
  7. 本番リリースする
    更新したデータは、本番リリース後に反映されます。
    →「本番リリースする」参照

バリューについて

バリューは以下の3種類があり、システムでの扱いが異なります。

  • 手動バリュー
    「車種の諸元情報」から判別できないバリューです。
    例) 「衝突安全テストで★5つ」など
  • 自動バリュー
    「車種の諸元情報」から判別できるバリューです。適用設定は自動的にスクリプトで行なわれます。手動で個別に設定することはできません。
    例) 「電気自動車」「小回りが利く」など
  • 自動バリュー (不可視)
    システムの内部処理に使用している特殊なバリューです。表側の検索項目としては表示されず、また設定もできません。

車種の手動バリューを変更する (対象が多い場合)

手動バリューの対象車種が多い場合は、CSV を介して行ないます。

  1. CSV ダウンロード画面を開く
    上部メニューの「CSV ダウンロード」をクリックします。
    「CSV ダウンロード」画面が開きます。
  2. CSV ファイルをダウンロードする
    「▼ CSV 取り込み用ファイルのダウンロード」の「出力タイプ」として「車種バリュー.csv」を選び、「作成」ボタンをクリックします。
    保存先とファイル名 (初期値は「車種バリュー.csv」) を指定し、CSV ファイルを保存します。
  3. CSV ファイルを更新する
    2 で保存した CSV ファイルを Excel などで開き、バリューを変更後、保存します。
  4. CSV 取り込み画面を開く
    上部メニューの「CSV 取り込み」をクリックします。
    「CSV 取り込み」画面が開きます。
  5. 取り込む CSV ファイルを指定する
    「▼ 本番リリース更新」にある「車種バリュー」の「ファイルを選択」ボタンをクリックします。
    表示されたファイル選択ダイアログで、4 で更新した CSV ファイルを指定します。
  6. CSV ファイルを取り込む
    「車種バリュー」の「取り込む」ボタンをクリックします。
    エラーが表示されなければ取り込みは成功です。
    エラーが出た場合は、表示されたエラー箇所を見直し、CSV ファイルを修正します。
  7. 本番リリースする
    更新したデータは、本番リリース後に反映されます。
    →「本番リリースする」参照

キャンペーンを設定する

  • 作業者 三菱オートリース様
  • 設定対象 一般 営業
  • 反映タイミング 即時

キャンペーンは「キャンペーン設定」画面より設定します。

  1. キャンペーン設定画面を開く
    上部メニューの「キャンペーン設定」をクリックします。
    「キャンペーン設定」画面が開きます。

  2. キャンペーンを登録する
    キャンペーンは3つまで登録できます。
    それぞれ、以下の項目を入力・登録します。

    • 「キャンペーン名」
      キャンペーン名を入力します。画像を登録しない場合、この文字列が表示されます。
    • 「遷移先 URL」
      リンク先を入力します (Marché のキャンペーン紹介記事など)。
    • 「キャンペーン画像」
      バナーとなる PNG 画像を登録します。適正サイズは横 250 ピクセル x 縦 90 ピクセルです。
      サイズが異なる画像を登録した場合、縦横比を保ったまま上記サイズに収まるよう、拡大または縮小表示されます。
      ブラウザによっては表示が荒くなることがありますので、適正サイズの画像を用意することをおすすめします。
  3. 更新する
    入力・登録を終えたら、「更新」ボタンをクリックします。
    内容は即時に反映されます (本番リリースは不要です)。

メンバー管理

  • 作業者 三菱オートリース様
  • 設定対象 営業
  • 反映タイミング 即時

メンバー権限について

メンバーの権限は下記の3種類あり、管理画面で利用できる機能が異なります。

  • システム管理者権限
    管理画面の全機能が利用できる権限です。
    「メンバー管理」画面では、全メンバーの改編ができます。
  • グループ管理者権限
    管理画面の一部機能が利用できる権限です。
    「メンバー管理」画面では、所属グループのメンバーのみ改編ができます。
  • 一般権限
    管理画面の利用ができない権限です。

新しいメンバーを登録する

新しいメンバーは、「メンバー管理」画面より設定します。

  1. 上部メニューの「メンバー管理」をクリックします。
    「メンバー管理」画面が開きます。
  2. 「新規追加」をクリックします。
    メンバー登録画面が開きます。
  3. メンバー情報を入力します。
    権限レベルは以下のように設定します。
    • システム管理者権限
      「所属グループ」：システム管理
      「権限レベル」：管理者
    • グループ管理者権限
      「所属グループ」：該当するグループ (システム管理以外)
      「権限レベル」：管理者
    • 一般権限
      「所属グループ」：該当するグループ (システム管理以外)
      「権限レベル」：一般
  4. 確認する
    内容を入力した後、「確認する」ボタンをクリックします。
    確認画面が表示されますので、内容に問題がなければ、「登録する」ボタンをクリックします。
  5. 登録完了
    登録完了画面が表示されます。
    登録したユーザーのパスワード有効期限は、登録日から90日となります。
    ※システム管理者ユーザーの場合、有効期限はありません。

パスワードの有効期限を更新する

パスワードの有効期限は、「メンバー管理」画面より更新します。

  1. 上部メニューの「メンバー管理」をクリックします。
    「メンバー管理」画面が開きます。
  2. メンバー詳細画面を開く
    設定したいメンバーを検索し、検索結果一覧の「詳細」ボタンをクリックします。
    「メンバー詳細」画面が開きます。
  3. メンバー編集画面を開く
    メンバー詳細画面の「編集」ボタンをクリックします。
    編集画面が開きます。
  4. パスワード有効日数を設定する
    「パスワード有効日数」のプルダウンを変更します。
    以下の場合は有効日数を変更できません
    • 権限がシステム管理者の場合
    • ステータスが無効の場合
  5. 確認する
    内容を入力した後、「確認する」ボタンをクリックします。
    確認画面が表示されますので、内容に問題がなければ、「更新する」ボタンをクリックします。
  6. 更新完了
    更新完了画面が表示されます。
